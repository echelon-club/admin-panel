# Echelon Admin Panel

A centralized admin dashboard for managing Echelon Club activities, events, and internal operations.

---

## 🚀 Overview

The Echelon Admin Panel is designed to streamline the management of club activities such as hackathons, workshops, member data, and platform operations.

This project is part of the Echelon ecosystem, focused on building real-world systems, improving collaboration, and creating scalable solutions within the student community.

---

## ✨ Features

* Secure admin authentication
* Dashboard with key insights
* Member management system
* Hackathon and event control
* Website content handling
* Scalable architecture for future expansion

---

## 🛠️ Tech Stack

## ⚙️ Getting Started

### 1. Clone the repository

```
git clone https://github.com/YOUR_ORG/admin-panel.git
cd admin-panel
```

### 2. Install dependencies

```
npm install
```

### 3. Run the development server

```
npm start
```

The application will run locally on:
[http://localhost:3000/](http://localhost:3000/)
